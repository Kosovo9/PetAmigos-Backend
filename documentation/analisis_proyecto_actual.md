- **Última actualización**: Diciembre 28, 2025

## Stack Tecnológico Actual

### Backend
- **Framework**: Node.js + Express.js
- **Base de Datos**: MongoDB con Mongoose
- **Tiempo Real**: Socket.io
- **Pagos**: Stripe (a eliminar según requerimientos)
- **IA**: OpenAI API
- **Autenticación**: JWT Authentication

### Frontend
- **Framework**: React 18
- **Build Tool**: Vite
- **Estilos**: Tailwind CSS
- **Animaciones**: Framer Motion
- **Iconos**: Lucide Icons
- **Cliente Tiempo Real**: Socket.io Client
